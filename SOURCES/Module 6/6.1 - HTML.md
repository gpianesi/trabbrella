
**HTML** is the standard language used to create documents that are displayed in web browsers.  
It is composed of **elements** delimited by **tags**, which form the fundamental building blocks of a website’s **structure** and **content**.

HTML is commonly used together with other web technologies, especially:

- **CSS** – presentation and layout  
- **JavaScript** – interactivity and behavior

---

# Evolution of HTML

HTML has gone through multiple major revisions since the early 1990s.

- **1991** – Tim Berners-Lee proposes **HTML 1.0**  
- **1993** – First official release of **HTML 1.0**  
- **1995** – **HTML 2.0**  
- **1997** – **HTML 3.0**  
- **1999** – **HTML 4.0**  
- **2000** – **XHTML 1.0**  
- **2008** – Introduction of **HTML5**  
- **2011** – Establishment of the **HTML Living Standard**

---

# HTML Syntax

## Core Concepts

- **Elements** – the primary building blocks (document nodes) used to structure web content.  
- **Tags** – mark the **beginning** and **end** of an element.  
- **Content** – text, images, links, or other material placed inside an element.  
- **Attributes** – provide additional information or properties for an element.

## Example

```html
<body>
  <p>This is a paragraph.</p>

  <img
    src="logo.png"
    width="400"
    height="200"
    alt="Website logo"
  />
</body>
```

---

# HTML Document Structure

A standard HTML page follows a clear hierarchy.

- **Document Type Declaration (DOCTYPE)** – specifies the HTML version.  
- **`<html>` element** – wraps the entire document.  
- **`<head>` element** – contains metadata and references to external resources.  
- **Meta tags** – define extra information such as description, charset, author, etc.  
- **`<title>` element** – sets the browser tab title.  
- **`<body>` element** – contains the visible page content.  
- **Content & semantic markup** – organize information in a meaningful, accessible way.

## Example Template

```html
<!DOCTYPE html>
<html>
<head>
  <meta name="description" content="Description of the website">
  <title>Website title</title>
</head>

<body>
  <div>
    <p>This is the website content</p>
  </div>
</body>
</html>
```
