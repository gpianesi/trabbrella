# LAMP Stack

The **LAMP stack** is a widely used web development software bundle consisting of:

- **Linux** – Operating System  
- **Apache** – Web Server  
- **MySQL** – Database Management System  
- **PHP** – Server-side Scripting Language  

Together, these technologies enable the development and deployment of **dynamic web applications**.

## Database Connectivity in PHP

To connect to a database, PHP uses:

- **Vendor-specific extensions** (e.g., `mysqli`, `pdo_mysql`)
- These extensions allow PHP to:
  - Establish connections
  - Execute queries
  - Retrieve results
  - Handle errors

---

# Connecting to a MySQL Database

PHP provides two main programming styles for MySQL interaction:

- **Procedural**
- **Object-Oriented (OOP)**

---

## 1. Procedural Approach

```php
<?php
$dbconn = mysqli_connect("localhost", "username", "password");

if (!$dbconn) {
    die("Connection error: " . mysqli_connect_error());
}

echo "Connection successful!";

// Query and response handling

mysqli_close($dbconn);
?>
```

### Explanation

- `mysqli_connect()` establishes a database connection.
- `mysqli_connect_error()` returns the error message if the connection fails.
- `mysqli_close()` terminates the connection.

---

## 2. Object-Oriented Approach

```php
<?php
$dbconn = new mysqli("localhost", "username", "password");

if ($dbconn->connect_error) {
    die("Connection error: " . $dbconn->connect_error);
}

echo "Connection successful!";

// Query and response handling

$dbconn->close();
?>
```

### Explanation

- `new mysqli()` creates a new database connection object.
- `$dbconn->connect_error` checks for connection errors.
- `$dbconn->close()` closes the connection.

---

# Querying a MySQL Database and Handling Results

## 1. Procedural Style

```php
<?php
$result = mysqli_query($dbconn, "SELECT * FROM table_name");

if (mysqli_num_rows($result) > 0) {
    while ($row = mysqli_fetch_assoc($result)) {
        echo "id: " . $row["id"] . " - Name: " . $row["name"] . "<br>";
    }
} else {
    echo "No results";
}

mysqli_close($dbconn);
?>
```

### Explanation

- `mysqli_query()` executes the SQL query.
- `mysqli_num_rows()` checks if any rows were returned.
- `mysqli_fetch_assoc()` retrieves rows as associative arrays.
- Results are processed inside a `while` loop.

---

## 2. Object-Oriented Style

```php
<?php
$result = $dbconn->query("SELECT * FROM table_name");

if ($result->num_rows > 0) {
    while ($row = $result->fetch_assoc()) {
        echo "id: " . $row["id"] . " - Name: " . $row["name"] . "<br>";
    }
} else {
    echo "No results";
}

$dbconn->close();
?>
```

### Explanation

- `$dbconn->query()` executes the SQL query.
- `$result->num_rows` checks the number of returned rows.
- `$result->fetch_assoc()` retrieves rows as associative arrays.
- The connection is closed after processing.

---

# Procedural vs Object-Oriented (Summary)

| Feature | Procedural | Object-Oriented |
|----------|------------|-----------------|
| Style | Function-based | Class-based |
| Syntax | `mysqli_function()` | `$object->method()` |
| Structure | Simpler for small scripts | Better for large applications |
| Maintainability | Less scalable | More modular and reusable |

---

# Key Takeaways

- The **LAMP stack** is a foundational web development environment.
- PHP supports multiple database systems beyond MySQL.
- MySQL interaction in PHP can be written in **procedural or OOP style**.
- Object-oriented style is generally preferred for **larger, maintainable applications**.
- Always close database connections after use.
