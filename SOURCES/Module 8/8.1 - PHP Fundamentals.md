
**PHP (Hypertext Preprocessor)** is an **open-source, server-side, interpreted scripting language** primarily used for **web development**. It can be embedded directly into HTML and is designed to generate **dynamic web content**.

PHP supports:

- **Dynamic content generation**
- **Session management**
- **Database interaction**
- **Form processing**
- **File handling**

> PHP is used by more than **75% of websites** that rely on a known server-side language.

---

## Basic Features of PHP

### 1. High-Level Programming

PHP provides common high-level programming constructs:

- **Variables**
- **Data types**
- **Functions**
- **Control structures** (e.g., `if`, `else`, `switch`, `loops`)

These features make PHP suitable for building structured and maintainable web applications.

---

### 2. Form Handling

PHP can process form data using:

- **GET** – sends data via URL parameters.
- **POST** – sends data within the HTTP request body.

This makes PHP ideal for:

- User input processing
- Authentication systems
- Contact forms
- Data submission workflows

---

### 3. File Operations

PHP supports file system interaction, including:

- Reading files
- Writing files
- Appending content
- File uploads

This functionality is essential for:

- Content management systems (CMS)
- Logging systems
- Data storage and retrieval

---

### 4. Session Management

PHP enables **session handling**, allowing user data to persist across multiple page requests.

Sessions are commonly used for:

- User authentication
- Shopping carts
- Personalized content
- Storing temporary user data

---

### 5. Database Interaction

PHP provides easy integration with popular databases such as:

- MySQL
- PostgreSQL
- SQLite
- Others via extensions (e.g., PDO)

This allows developers to:

- Connect to databases
- Execute queries
- Retrieve and manipulate data
- Build dynamic, data-driven websites

---

# Integrating PHP with HTML

PHP can be integrated with HTML in two main ways:

## 1. Using Separate `.php` Files

- Files with the `.php` extension are processed by the **PHP interpreter** on the server.
- The interpreter generates **HTML output**, which is then sent to the client's web browser.
- The browser only receives the final HTML — **PHP code is never visible to users**.

---

## 2. Embedding PHP Within HTML

PHP code can also be embedded directly inside an HTML file.

This approach allows:

- Dynamic content insertion
- Variable usage inside HTML
- Conditional rendering of content
- Cleaner separation of logic and presentation (when structured properly)

---

# Example: Pure PHP Output

```
<?php
$greeting = "Hello, welcome to the website!";

echo "<html><head><title>Greeting</title></head>";
echo "<body><h1>$greeting</h1></body>";
echo "</html>";
?>
```

### Explanation

- A variable `$greeting` stores a string.
- `echo` outputs HTML content.
- The PHP interpreter generates a complete HTML page.

---

# Example: Embedding PHP in HTML

```
<!DOCTYPE html>
<html>
<head>
    <title>Web Page Title</title>
</head>
<body>

<?php
$currentTime = date('h:i:s a');
echo "<p>Current time is: $currentTime</p>";
?>

</body>
</html>
```

### Explanation

- Standard HTML structure is written normally.
- PHP is embedded using `<?php ... ?>` tags.
- The built-in `date()` function generates the current time.
- The result is dynamically inserted into the HTML page.

---

- **It generates dynamic HTML**.
- **It integrates seamlessly with HTML**.
