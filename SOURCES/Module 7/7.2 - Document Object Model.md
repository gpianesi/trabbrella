
The **Document Object Model (DOM)** represents a web page as a **tree structure of nodes**.

- Each HTML element is a **node** in the tree.
- Nodes can be **accessed, modified, added, or removed** using JavaScript.
- The DOM acts as an interface between **HTML documents** and **JavaScript**.

Through the DOM, JavaScript can:

- Change **HTML content**
- Modify **CSS styles**
- Add or remove elements
- Update element attributes
- Respond to user interactions

---

# Selecting Elements

Elements can be selected using:

- **ID**
- **Class**
- **Tag name**
- **CSS selectors**

## Common DOM Selection Methods

| Method | Description | Returns |
|--------|------------|----------|
| `getElementById()` | Selects an element by its ID | Single element |
| `getElementsByClassName()` | Selects elements by class name | HTMLCollection |
| `getElementsByTagName()` | Selects elements by tag name | HTMLCollection |
| `querySelector()` | Selects the first element matching a CSS selector | Single element |
| `querySelectorAll()` | Selects all elements matching a CSS selector | NodeList |

## Examples

```javascript
var element = document.getElementById('text');

var elements = document.getElementsByClassName('normal');

var elements = document.getElementsByTagName('p');

var element = document.querySelector('div.description');

var elements = document.querySelectorAll('p.normal_text');
```

> **Note:**  
> - `getElementsByClassName()` and `getElementsByTagName()` return live collections.  
> - `querySelectorAll()` returns a static NodeList.

# Modifying Elements

JavaScript allows dynamic manipulation of DOM elements.

## 1. Creating Elements

Use `createElement()` to create a new node.

```javascript
var newElement = document.createElement("p");
newElement.textContent = "This is a new paragraph.";
document.body.appendChild(newElement);
```

---

## 2. Removing Elements

Use `removeChild()` to remove a node from its parent.

```javascript
var elementToRemove = document.getElementById("elementId");
elementToRemove.parentNode.removeChild(elementToRemove);
```

---

## 3. Modifying Content and Styles

### Change Text Content

```javascript
document.getElementById("elementId").textContent = "Updated text content.";
```

### Change CSS Style

```javascript
document.getElementById("elementId").style.color = "blue";
```

---

## 4. Modifying Attributes

Use `setAttribute()` to update element attributes.

```javascript
document.getElementById("elementId")
        .setAttribute("href", "https://www.example.com");
```

> **Commonly Modified Attributes:** `href`, `src`, `class`, `id`, `value`

# Event Handling

## What Are Events?

An **event** is an action or occurrence detected by the browser, such as:

- Mouse clicks
- Key presses
- Form submissions
- Window resizing
- Touchscreen interactions
- Media events
- Drag and drop actions

JavaScript handles events using **event listeners**.

---

## Event Listeners

The `addEventListener()` method attaches a function that executes when a specified event occurs.

### Syntax

```javascript
element.addEventListener('event', function() {
  // code to execute
});
```

---

## Example: Click Event

```html
<!DOCTYPE html>
<html>
<head>
  <title>Event Listener</title>
</head>
<body>

<button id="clickMeButton">Click Me</button>

<script>
  var button = document.getElementById('clickMeButton');

  button.addEventListener('click', function() {
    alert('Button was clicked!');
  });
</script>

</body>
</html>
```

### Explanation

1. The button is selected using `getElementById()`.
2. An event listener is attached for the `'click'` event.
3. When the button is clicked, the function executes.

# Summary

The **DOM** enables JavaScript to dynamically control web pages by:

- Selecting elements
- Creating and removing nodes
- Modifying content, styles, and attributes
- Handling user interactions through events

DOM manipulation is fundamental to building **interactive and dynamic web applications**.
